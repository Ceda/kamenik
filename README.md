# Kamenictvi Rejda
 Sass/coffe starter pack for kamenictvi rejda

## Grunt
### Install grunt
```bash
$ npm install in libs/ directory
```
### Run Grunt
```bash
$ cd {projectRoot}/libs
$ grunt
```
